# GitHub Pages启用步骤

当前仓库：https://github.com/leezhihua-mem/zhongyou-insurance-ui

## 启用步骤

1. 打开仓库Settings页面：
   https://github.com/leezhihua-mem/zhongyou-insurance-ui/settings/pages

2. 在"Source"部分：
   - Branch: 选择 "main"
   - Folder: 选择 "/ (root)"
   
3. 点击"Save"

4. 等待1-2分钟，页面会自动部署

5. 访问地址：
   https://leezhihua-mem.github.io/zhongyou-insurance-ui
